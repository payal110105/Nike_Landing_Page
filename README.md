# Nike Landing Page

A modern, responsive landing page for Nike, designed with a clean, minimalistic approach. This page utilizes Tailwind CSS to create a visually appealing layout that works seamlessly across all screen sizes. The design focuses on bold typography, high-quality images, and a user-friendly experience.

## Features
- **Responsive Design**: Fully responsive across mobile, tablet, and desktop devices.
- **Tailwind CSS**: Fast, utility-first CSS framework to streamline development and styling.
